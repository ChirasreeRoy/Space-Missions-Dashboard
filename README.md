# 🚀 GLOBAL SPACE MISSIONS ANALYSIS

## 🚀1. Project Title

**Global Space Missions Analysis (1957–2022) — End-to-End Data Analytics Project (SQL & Power BI)**
Analyzing historical space missions to uncover trends in launches, mission success, country dominance, and global space infrastructure.

## 🎯2. Purpose Of The Project

This project focuses on analyzing global space mission data using an end-to-end data analytics approach.

* SQL was used for data cleaning, preprocessing, and feature engineering to prepare the dataset for analysis.
* SQL was further utilized to perform data analysis, aggregation, and create structured views for reporting.
* Power BI was used to build an interactive dashboard that visualizes launch trends, mission success rates, country performance, and geographic patterns.

The purpose of this project is to transform raw space mission data into meaningful insights that highlight the evolution of the space industry and support analytical decision-making.

## 🛠️3. Tech Stack

* 🗄️ **SQL Server** → Data Cleaning, Transformation, Feature Engineering, Analytical Queries
* 📊 **Power BI** → Data Visualization, Dashboard Creation, Interactive Reporting
* ⚙️ **DAX (Data Analysis Expressions)** → KPI Measures (Total Missions, Success Rate, etc.)
* 📝 **Data Visualization Techniques** → KPI Cards, Line Charts, Bar Charts, Map Visual, Stacked Column Charts, Slicers

## 📂4. Data Source

* 📡 **Global Space Missions Dataset (Next Spaceflight)**

Contains mission-level information such as:

* 🚀 Rocket details and status
* 🌍 Launch location and country
* 📅 Date and time of launch
* 📊 Mission outcome (Success/Failure)
* 💰 Mission cost (where available)

## ✨5. Features and Highlights

### 💼a) Business Problem

The global space industry has evolved rapidly, but understanding trends in mission success, launch frequency, country dominance, and infrastructure distribution is complex due to unstructured and raw data.

Organizations and analysts often struggle to identify:

* How space missions have evolved over time
* Which countries dominate space exploration
* Which rockets are most reliable and widely used
* Where launch activities are geographically concentrated

Without proper analysis, it is difficult to extract meaningful insights from historical mission data.

### 🎯b) Goal of the Project

The goal of this project was to build a complete analytics solution to understand space mission trends and support data-driven insights.

* 🗄️ **SQL:**
  Used to clean the dataset, handle missing values, and engineer features such as launch year, decade, country, and launch site.
  It was also used to create analytical views for trend analysis, success rate evaluation, and segmentation.

* 📊 **Power BI:**
  Used to build a 2-page interactive dashboard that presents insights visually, enabling users to explore mission trends, performance, and geographic patterns efficiently.

### 🔄c) Project Walkthrough

#### 🔵 SQL (Data Cleaning & Analysis)

* ✔️ Removed duplicates and handled missing values
* ✔️ Extracted key features:

  * Launch Year
  * Launch Decade
  * Country
  * Launch Site

Created analytical views:

* 📈 Launches per Year
* 📊 Success Rate over Time
* 🌍 Country-wise Successful Missions
* 🚀 Rocket Usage and Status
* 📍 Launch Site Analysis

Used:

* Aggregations (COUNT, SUM)
* Conditional logic (CASE WHEN)
* Grouping and filtering for insights

##### 📊 Page 1: Overview Dashboard

* 📌 KPI Cards → Total Missions, Successful Missions, Success Rate
* 📈 Line Chart (Launch Trends) → Shows how launch frequency evolved over time
* 📈 Line Chart (Success Rate) → Tracks improvement in mission reliability
* 📊 Bar Chart (Country Success) → Identifies top-performing countries
* 📊 Bar Chart (Rocket Usage) → Shows most used rockets and their status
* 🌍 Map Visual → Displays global distribution of space launches
* 📊 Bar Chart (Launch Sites) → Highlights top launch locations

##### 📊 Page 2: Key Insights

* 📄 Summary of insights for stakeholders

* 🧭 Designed for executive-level understanding

### 🚀 Dashboard Features

* 🔄 Interactive filters (Year, Country)

* 📊 Consistent design and theme
* 💡 Separate insights page for business interpretation

### 📈 d) Business Impact & Insights

#### 🧠 Key Insights

* 🚀 Launch frequency has increased significantly since the early 2000s
* 📈 Mission success rates have improved steadily over time
* 🌍 USA and Russia historically dominate space missions, with China emerging strongly
* 🛰 A small number of rockets account for a large portion of missions
* 📍 Launches are concentrated in key locations like Cape Canaveral and Baikonur

#### 📌 Business Impact

* 📊 Provides a clear understanding of historical trends in the space industry
* 🚀 Helps identify technological progress and mission reliability improvements
* 🌍 Highlights global competition and emerging space powers
* 📍 Identifies key launch infrastructure and geographic clusters
* 📈 Enables strategic insights for planning future space missions and investments

## 🏁6. Conclusion

This project demonstrates how historical space mission data can be transformed into meaningful insights using SQL and Power BI. It highlights trends in launch activity, mission success, and global participation, providing a clear view of the evolution of the space industry.
